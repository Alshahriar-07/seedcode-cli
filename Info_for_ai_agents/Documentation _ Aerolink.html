Install Claude Code

A walk-through for installing the Claude Code client — from a clean machine to your first claude command.

01

Install Node.js (if you have not already)

The CC client requires Node.js 18 or newer. First, check what's installed:

bash
node --version

If the printed version is ≥ 18.0.0, skip ahead. Otherwise install via the official installer or Homebrew:

bash
brew install node
02

Create an API key

Open the API Keys page, click Create key, give it a name, and copy the secret.

Treat your key like a password. Anyone with it can spend your credits.

03

Install the CC client

bash
npm install -g @anthropic-ai/claude-code

If the command is not found after install, restart Terminal.

04

Create a settings.json file

Write your config to ~/.claude/settings.json:

json
{
    "env": {
        "ANTHROPIC_API_KEY": "YOUR_API_KEY",
        "ANTHROPIC_BASE_URL": "https://capi.aerolink.lat/",
        "CLAUDE_CODE_DISABLE_NONESSENTIAL_TRAFFIC": "1"
    },
    "permissions": {
        "allow": [],
        "deny": []
    },
    "apiKeyHelper": "echo 'YOUR_API_KEY'"
}

Replace both YOUR_API_KEY with the key from step 2.

05

Launch the client

bash
claude

You should see the welcome banner and a prompt. You're all set.

FAQ

How do I update?

Re-run the install command:

bash
npm install -g @anthropic-ai/claude-code

How do I uninstall?

Run:

bash
npm uninstall -g @anthropic-ai/claude-code
