## Code Style

PEP8

Type hints

Small functions

Meaningful names

Comments only when necessary

---

## Error Handling

Never crash.

Catch exceptions.

Display readable messages.

Retry network errors.

---

## Security

Never upload API keys.

Never log API keys.

Store configuration locally.

---

## Performance

Fast startup.

Lazy loading.

Minimal memory usage.

---

## User Experience

Friendly messages.

Beautiful terminal.

Responsive.

Streaming output.

---

## Don't

Don't hardcode API keys.

Don't require backend servers.

Don't collect user data.

Don't send analytics.

Don't use unnecessary dependencies.
